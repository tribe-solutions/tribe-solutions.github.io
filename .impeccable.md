# Impeccable

## Design Context

### Users
Dois públicos: (1) donos de negócio e gestores brasileiros buscando software sob medida (web, mobile, bots WhatsApp) — chegam pelo Google ou indicação, decidem rápido, conversão é abrir conversa no WhatsApp; (2) gestores de FAPs e órgãos públicos avaliando a Tribe como fornecedora via licitação — buscam credibilidade, histórico (40+ licitações vencidas) e formalidade (ICP-Brasil).

### Brand Personality
"Engenharia séria, feita no calor da floresta." Três palavras: competente, amazônica, direta. Rigor técnico de big tech com identidade regional autêntica (Manaus → Brasil). Não é agência genérica de SP; a floresta é diferencial, não enfeite.

### Aesthetic Direction
Tema claro editorial-orgânico: fundo papel (#f8f6f0) com grain, verde-floresta profundo + dourado como acento raro, folhas amazônicas SVG em sway lento no hero, seção escura (forest-900) para licitações. Display serif (hoje Fraunces) + sans de corpo (hoje Inter). Anti-referência: dark mode com glow neon, glassmorphism, template de SaaS americano.

### Design Principles
1. A floresta aparece com sutileza e movimento lento — nunca como clipart.
2. Toda seção empurra para uma conversa no WhatsApp; um CTA primário por viewport.
3. Dourado é raro: só para licitações/conquistas.
4. Credibilidade por especificidade (números, nomes de fundações, endereço real), não por adjetivos.
5. Motion suave e respeitoso: ease-out, prefers-reduced-motion sempre.
